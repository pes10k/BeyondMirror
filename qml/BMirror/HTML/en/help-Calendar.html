Calendar

The Calendar widget, like all widgets, has the following features:

  • Edit button in the top right, which brings down the widget's configuration panel
  • Title of the widget is in the center top of the widget
  • Close button in the top left of the widget's window, which closes the widget
  • Draggability / positioning: By tapping and dragging on a non-input part of a widget, users can position it wherever they'd like on the device's face

The Calendar widget displays a list of events. The format of an event includes the name of the event, the data and the time. The envents of calendar is retrived from an online source.

In the configuration panel, users can retrive their google calendars by logging into their google accounts.
