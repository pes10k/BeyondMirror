Setting

Setting widget allows user to reconfigure the language and the wireless connection The two configuration are seperated by their tabed names.

Language

The supported languages appear in a list that the user can scroll through. Selecting a language updates all interface text immediately.

Wireless

All automatically detected wireless networks will appear in the list labeled 'WiFi'. If the user would prefer to connect to a wireless network that is not broadcasting their SSID, they can enter the name of the preferred network in the text field labeled 'Other network name'. Once the user has selected the network they would like to connect the mirror to, they will need to input the wireless network password. Once all fields have been filled, the device will attempt to connect to the network to verify the settings are correct.
